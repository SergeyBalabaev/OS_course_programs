# Программа для работы с переключающимися светодиодами

Сборка: 
``` 
gcc -o led_blinker led_blinker.c  
```

Запуск:  
sudo ./led_blinker [-h][-q] T   
-h - описание работы  
-q - тихий режим  
T - время задержки (сек)

Пример: 
```
sudo ./led_blinker -q 10 //Переключение светодиода каждые 10 секунд
```
